# Riff Launch Pricing Strategy

## 목적
- 출시 초기 6개월 동안 사용자 확산과 유료 전환을 동시에 노린다.
- 개인 생산성 사용자와 프리랜서를 첫 핵심 고객으로 가정한다.
- 몇 달 무료 대신, 상시 무료 플랜과 짧은 Pro 체험으로 지불 의사를 빠르게 검증한다.

## 출시 권장안
- `Free + 14일 Pro 체험`을 기본 구조로 사용한다.
- 가입 시 신용카드 없이 `14일 Pro 전체 기능`을 연다.
- 체험 종료 후에는 계정을 막지 않고 `Free`로 자동 다운그레이드한다.
- 런칭 3~6개월 동안은 얼리어답터 가격 `월 $7.99 / 연 $69`를 유지한다.
- 제품 가치가 검증되면 신규 가입자 기준 `월 $9.99~12.99`까지 인상 여지를 남긴다.

## 무료 플랜 경계
무료 플랜은 제품을 실제로 계속 쓰게 만드는 수준까지 열어둔다.

### Free에 남길 핵심 가치
- 주간 캘린더 중심의 기본 일정 관리
- 루틴, 할일, 일기, 감정 기록
- 기본 캘린더 통합과 가벼운 개인 사용

### Pro로 올릴 경계
- 고급 뷰와 필터
- 고급 커스터마이징
- 확장 연동 수와 자동화
- 향후 AI/스마트 플래닝 기능

## 체험 전환 흐름
1. 사용자가 가입하면 즉시 Pro 기능을 14일 동안 사용한다.
2. 체험 중에는 고급 기능이 실제 습관에 어떤 차이를 만드는지 보여준다.
3. 체험 종료 후에는 Free로 자동 전환하고, 데이터와 기본 사용 흐름은 유지한다.
4. 업그레이드는 특정 제한에 닿는 순간 자연스럽게 제안한다.

## 메시지 원칙
- 무료는 미끼가 아니라 실제 사용 가능한 기본 제품이어야 한다.
- Pro는 "없으면 못 쓰는 기능"보다 "더 깊게 쓰는 사람에게 필요한 기능"으로 보여야 한다.
- 가격 카피는 복잡한 할인 구조보다 `명확한 월간 가격 + 큰 연간 절감`이 낫다.

## 런칭 직후 핵심 지표
- 첫 3일 내 캘린더 연결률
- 첫 이벤트 생성률
- 첫 주간 뷰 진입률
- 14일 체험 시작 대비 유료 전환율
- 무료 사용자 4주 유지율
- 유료 전환 시 어떤 제한에 닿았는지

## 이벤트 제안
마케팅/가격 관련 프론트 이벤트는 다음 이름으로 수집한다.

- `marketing_cta_clicked`
- `pricing_cta_clicked`
- `pricing_section_viewed`
- `trial_flow_viewed`

## 구현 반영 범위
- 랜딩 페이지에 출시 가격 섹션을 노출한다.
- Free / Pro 경계와 14일 체험 흐름을 사용자 카피로 명확히 보여준다.
- CTA와 가격 섹션 조회를 추후 분석 도구로 연결할 수 있도록 추적 이벤트를 남긴다.
